Gerador de Ordem de Serviço

Este projeto tem como objetivo a criação de um Gerador de Senha Aleatória com a possibilidade de alternar entre tema claro e tema escuro. O usuário pode personalizar a senha gerada escolhendo diferentes opções de caracteres (números, símbolos, maiúsculas, minúsculas, etc.) e também pode visualizar a senha gerada com o nível de complexidade.
Projeto desenvolvido em HTML, CSS e JavaScript.

API de Ponto Eletrônico

Esta é uma API RESTful para registro e consulta de pontos eletrônicos de funcionários. Desenvolvida com Flask e Flask-RESTX, a API oferece funcionalidades para registrar pontos de entrada e saída, consultar pontos de um usuário específico, atualizar e excluir registros de ponto, além de fornecer documentação interativa usando o Swagger.
Tecnologias Utilizadas:
Flask - Framework web para Python.
Flask-RESTX - Extensão do Flask para criar APIs RESTful.
SQLite - Banco de dados relacional leve.
Swagger - Interface interativa para documentação da API.
